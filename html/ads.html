Análise e Desenvolvimento de Sistemas

Curso tecnólogo focado na prática do desenvolvimento de sistemas, com entrada mais rápida no mercado de trabalho.

Sobre o Curso

Objetivo

Formar desenvolvedores práticos capazes de analisar, projetar e desenvolver sistemas de informação de forma eficiente.

Perfil do Estudante

Ideal para quem quer entrar rapidamente no mercado de trabalho como desenvolvedor, com foco prático e menos teoria.

Principais Diferenciais

  • Curso mais curto e prático
  • Foco em desenvolvimento
  • Entrada rápida no mercado
  • Menos matemática e teoria
  • Mais laboratórios práticos

  • Lógica de Programação
  • Banco de Dados
  • Programação Orientada a Objetos
  • Interface Humano-Computador
  • Segurança da Informação
  • Desenvolvimento Web
  • Análise de Sistemas
  • Desenvolvimento Mobile
